# Python Automation Toolkit
This repository contains a set of lightweight Python scripts designed for everyday automation tasks.

## Scripts included:
- **File Organizer**: Sorts your files into folders based on their extensions.
- **Web Scraper**: Extracts headlines from news websites.
- **Data Cleaner**: Cleans CSV files by removing duplicates and empty rows.

## How to use:
All scripts require Python 3. Run them via terminal:
`python3 <script_name>.py`
